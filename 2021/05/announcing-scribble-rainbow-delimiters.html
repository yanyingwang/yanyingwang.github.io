Announcing scribble-rainbow-delimiters

2021-05-31 / racket, package, english

Recently I extract my code from a former project rainbow-delimiters to scribble-rainbow-delimiters, which is useful for reading code on HTML page generated by Racket’s Scribble.

If you impose this package to you scribble code, then the generated HTML pages would include js scripts to enable you highlighting the matched brackets of code, check this package’s README and doc for the detail of this.

And in my opinion, I’d better create a PR to merge my code to Racket’s Scribble. But since I’m not fimiliar with its code and I’m not sure enough for the needing of this for Racket’s Scribble, so I choose to leave this for the future for me to deal with.
